# Custom Set Data Structure in C++

A C++ implementation of the Set Abstract Data Type (ADT) using dynamic arrays, supporting fundamental set operations while maintaining unique elements.
## Features
- Insert and remove elements
- Cardinality calculation
- Union
- Intersection
- Symmetric Difference
- Copy Constructor
- Destructor

## Concepts Used
- Object-Oriented Programming (OOP)
- Dynamic Memory Allocation
- Arrays
- Set ADT
